情報基礎１
B4001 FUNDAMENTALS OF INFORMATION TECHNOLOGY 1(01) （Spring ） >> View Syllabus
Faculty-in-chargeTagiru Nakamura
Class TypeLecture, Workshop
Semester, Day and PeriodWednesday 4th Period,Wednesday 5th Period
Language used in the class Japanese

In SFC, all the students are expected to utilize the web system as infrastructure for study, research, production, and transmission. In order to do that, you need a capability not only to see other people's work but to create original contents using the web,

In this course, we first learn fundamental knowledge of computers, as well as programming skills which are necessary to the modern web system. After that, we make web pages by using the knowledge and skills learned.

After this course, you will learn advanced topics about web, network and programming in Fundamentals of Information Technology 2.

SFCではウェブを全学生の知的共通基盤とし、ウェブを活用して学習・研究・制作・発信を行えるようにしたいと考えています。そのためには、単にウェブで他人の作ったものを「見る」だけでなく、ウェブを使って新しいものを「創る」能力が必要です。

この科目では、まず前提となるコンピュータの基本的な仕組みを学び、それと並行して、ウェブを使いこなすために必要となるプログラミングなどの技術を学びまず。そして、それらの知識と技術を応用してウェブページの制作を行います。

なお、「情報基礎２」では、この科目の内容を基礎としてウェブ、ネットワーク、プログラミングについてさらに進んだ内容を学びます。

クラスという概念を使ってプログラムを部品としてまとめ、その部品の動作を組み合わせることがオブジェクト指向プログラミング（OOP:Object Oriented Programming）です。プログラムは「クラス」という概念を使わずに書くこともできますが、プログラムを書く際に自分が分かりやすく・他人に読んでもらいやすくするために「クラス」を使ってプログラムを書きます。 オブジェクト指向プログラミングをマスターすると、他の人が作ってくれた部品：クラスを組み合わせて、便利なプログラムをさっと作ったり、大きなプログラムを一人で作ることができるようになります。

この授業では、OOPに必要な概念の勉強と実際にプログラムを書く練習を繰り返し、OOPを習得します。 OOPは様々な言語で使われているプログラミングの方法ですが、この授業ではJava言語のサブセットであるProcessingを使ってOOPを習得します。 後半にはEclipseを使ってJavaのプログラムからProcessingのクラスを呼び出し、より本格的なプログラミングへと発展させます。
